# 闲鱼跟单助手 (Xianyu Scout)

用 DeepSeek + 网页抓取，做闲鱼信息差生意。搜品 → 比价 → 识别捡漏 → 生成文案。

## 功能

- **搜品分析**：搜闲鱼某类商品，拉出价格分布，标出最低价和高频价区间
- **捡漏识别**：AI 判断哪些商品价格明显低于市场均值，抓出来
- **自动写文案**：DeepSeek 生成闲鱼商品标题+描述，带热搜词
- **跨平台比价**：同一商品，闲鱼 vs 拼多多/淘宝搜同款，算差价

## 用法

在 OpenClaw 对话中直接说：

```
帮我搜闲鱼上 "机械键盘" 的价格分布
```

```
这个闲鱼链接能捡漏吗：https://m.tb.cn/xxx
```

```
帮我给这个商品写闲鱼文案：二手 iPhone 14 128G 黑色 9成新
```

```
查一下 "蓝牙耳机" 闲鱼和拼多多的差价
```

## 安全警告

- 闲鱼网页版有反爬限制，优先用公开搜索页面抓取
- 不模拟登录，不操作账号，只读搜索
- 捡漏建议仅为 AI 判断，实际交易风险自负
- 不要用此工具发送垃圾消息或骚扰卖家
- API Key 不要暴露在聊天中

## 依赖

- Python 3.8+（用于 requests + BeautifulSoup 抓取）
- DeepSeek API（已在 OpenClaw 中配置）
