# 闲鱼爆款文案生成器 (Xianyu Copywriter)

用 DeepSeek 的文案能力，为闲鱼商品批量生成爆款标题和描述。不爬数据，纯 AI 驱动，100% 可用。

## 功能

- **爆款标题生成**：输入产品名或关键信息，生成 5-10 条带热搜词的标题
- **高转化描述**：自动写卖点突出的商品描述，含信任背书和行动号召
- **关键词优化**：注入闲鱼搜索高频词，提高曝光
- **定价建议**：基于品类知识给出参考定价区间
- **标签生成**：自动生成搜索标签

## 用法

在 OpenClaw 对话中直接说：

```
帮我写闲鱼文案：二手 iPhone 14 128G 黑色 9成新 电池95%
```

```
给这个产品生成 5 个标题：Cherry 青轴机械键盘 87键 黑色
```

```
优化我的闲鱼描述：[粘贴现任描述]
```

```
分析并优化：这款蓝牙耳机标 ¥89 卖得出去吗？
```

## 最佳实践

1. 描述里加数字——"电池95%""用了3个月"比"基本全新"可信
2. 标题最前面放卖点词：🔥/💰/⚡ + 品牌 + 型号
3. 标签里带品类词，方便搜索抓取
4. 标价留 5-10% 砍价空间，买家砍一刀就有成交感
5. 标明"支持面交""提供验机图"降低买家顾虑
