# 闲鱼爆款文案生成器 (Xianyu Copywriter)

> ✍️ ClawHub 首个中国电商 AI 文案 Skill —— 用 DeepSeek 批量生成闲鱼爆款文案，提高转化率。

## 功能

- 🏷️ **多标题生成** — 输入产品信息，输出 5-10 条爆款标题
- 📝 **高转化描述** — AI 写卖点突出、信任感强的商品描述
- 🔑 **关键词优化** — 注入闲鱼热搜词，提高搜索曝光
- 💰 **定价建议** — 基于品类知识的参考定价区间
- 🏷️ **标签生成** — 自动生成搜索标签

## 为什么用这个

- 闲鱼 3 亿用户，好文案就是好销量
- 自己写半天想不出好标题？AI 30 秒出 5 个版本
- 不需要爬虫、不需要代理、不需要 Cookie——纯 AI 驱动，100% 可用
- 中国市场上 ClawHub 唯一的本地电商文案 Skill
