# 闲鱼跟单助手 (Xianyu Scout)

> 🐟 ClawHub 首个中国电商市场 Skill —— 用 AI 做闲鱼信息差生意。

## 功能

- 🔍 **闲鱼搜品** — 搜任何品类，拉出价格分布
- 🎯 **捡漏识别** — AI 判断低于市场价的商品
- ✍️ **自动写文案** — DeepSeek 生成高转化标题+描述
- 📊 **跨平台比价** — 闲鱼 vs 拼多多，找搬砖机会

## 安装

```bash
clawhub install xianyu-scout
```

或者在 OpenClaw 中：
```
openclaw skills install xianyu-scout
```

## 使用示例

```
帮我搜闲鱼上 "蓝牙耳机" 的价格，找捡漏
```

```
给这个商品写闲鱼文案：二手 MacBook Air M1 8G+256G 银色 95新 电池循环50次
```

```
闲鱼上 "乐高机械组" 和拼多多比价，差价多少？
```

## 为什么用这个 Skill

- 闲鱼 3 亿用户，每天产生海量信息差
- 靠人肉翻页找低价，效率太低
- 这个 Skill 让 OpenClaw 自动扫、自动判断、自动出文案
- 中国市场上 ClawHub 唯一的本地电商助手

## 依赖

- Python 3.8+
- DeepSeek / OpenAI API（在 OpenClaw 中配置）
