## 闲鱼商品文案最佳实践

### 标题公式
[卖点符号] + [品牌] + [型号] + [关键参数] + [成色] + [保障]

### 高转化标题技巧
1. 用 🔥💰⚡📱🎯 等符号吸引眼球
2. 核心关键词放最前面：品牌 + 型号
3. 加数字增加可信度：电池95%、只用了3个月
4. 制造紧迫感：急出、秒杀价、限量
5. 明确保障：顺丰包邮、支持面交、可小刀

### 描述结构
1. 第一句：品牌 + 型号 + 成色（一句话说清楚）
2. 第二句：使用情况（用数字，不用说"很好"）
3. 第三句：包含什么（配件、包装）
4. 第四句：交易方式（面交/邮寄，降低顾虑）
5. 结尾：标签 #tag1 #tag2 #tag3

### 定价策略
- 新品二手 8-9 成新：原价 60-70%
- 新品二手 6-7 成新：原价 40-50%
- 电子数码留 5-10% 砍价空间
- 日用品留 10-15% 砍价空间
- 稀有/绝版品可以溢价 10-30%

4. 标签别超过 5 个，选精准词
