# 闲鱼跟单助手 — OpenClaw 详细执行指南

## 触发词

当用户提到以下关键词时加载此 Skill:
- 闲鱼、xianyu、闲鱼搜索、闲鱼比价、闲鱼捡漏
- 闲鱼文案、闲鱼描述、商品描述优化
- 信息差、搬砖、选品、跟单

## 执行流程

### 1. 搜索闲鱼商品

使用 `web_fetch` 搜索闲鱼:

```
web_fetch url="https://s.2.taobao.com/list/list.htm?q=<关键词>&search_type=item"
```

从返回的 HTML 中提取:
- 商品标题 (`.item-title` 或 `.J_ItemData` 区块)
- 价格 (`.price` 元素)
- 卖家位置
- 浏览量/想要数

### 2. 价格分析

用以下逻辑计算:
- 最低价、最高价、中位数、均价
- 价格区间分布 (低/中/高)
- 捡漏: 价格 < 均价 * 0.6 的商品

### 3. 文案生成

用以下模板让 DeepSeek 生成文案:

```
你是闲鱼爆款文案专家。根据以下商品信息，生成一条闲鱼商品文案：

商品：{商品名}
状态：{新旧程度/成色}
卖点：{特殊卖点}

要求：
1. 标题15-25字，包含热搜关键词
2. 描述3-5句，突出卖点
3. 加3-5个话题标签
4. 语气真诚自然，不要过于营销
5. 用 emoji 点缀关键信息

输出格式：
**标题**：...
**价格**：推荐 ¥...
**描述**：...
**标签**：#xxx #xxx
```

### 4. 跨平台比价

如果需要比价:
1. 搜索拼多多同款: `web_fetch url="https://mobile.yangkeduo.com/search_result.html?search_key=<关键词>"`
2. 对比闲鱼最低价 vs 拼多多最低价
3. 如果差价 > 30%，标记为搬砖机会

## 输出格式

每次查询返回以下 Markdown:

```markdown
## 🐟 闲鱼搜品: 《商品名》
- 共 N 个结果
- 最低：¥X | 最高：¥Y | 中位数：¥Z
- 📊 价格分布图 (文字版)

### 🔍 捡漏候选
1. [商品] ¥X | 低于市场 Y%

### ✍️ 推荐文案
[生成的标题+描述]

### 🚚 搬砖机会 (如有)
拼多多 ¥X vs 闲鱼 ¥Y | 差价 Z 元
```

## 限制与安全

- 闲鱼网页版反爬严格，建议每次搜 1-2 页
- 不要高频调用（间隔至少 30 秒）
- 捡漏判断仅供参考，不构成投资建议
- 不要用此工具操作闲鱼账号或自动上架
